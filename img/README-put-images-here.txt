Put product images in this folder.

The website reads images from this folder when you run:
node server.js

Useful filename examples:
- shirt-vintage.jpg
- converse-red-shoes.jpg
- denim-jeans-levis.jpg
- cap-denim.png
- bag-tote.webp
- belt-brown.jpg
- sunglasses-black.jpg

If a filename does not match a category keyword, it will appear in Other.
